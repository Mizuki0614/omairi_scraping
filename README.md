# omairi_scraping

全国の神社仏閣の情報をomairi.comからのデータ抽出


omairi.com(https://omairi.club/)での

[名称]

[住所]

[寺社人気ランキング（都道府県）]

[寺社人気ランキング（全国）]

[アクセス数]

[写真数]

[電話番号]

[URL]

[御朱印の有無]

[御朱印写真のURL]

の摘出


・CSV形式でのデータの保存(omairi_crawler_csv)

・DB(MySQL, DB='omairi', user='fine', passwd='fine', charset='utf8mb4')形式でのデータの保存(omairi_crawler_mysql.py)
